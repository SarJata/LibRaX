# LibRaX Project

Welcome to the LibRaX project repository. This folder contains both the ready-to-use application and the complete source code.

## Folder Structure

*   **`Application/`**: This folder contains the compiled Android Package Kit (APK) file.
    *   `LibRaX (Version 1.0).apk`: The installable app file.
*   **`Source Code/`**: This folder contains the complete Android Studio project source code.
    *   Developers can open this folder in Android Studio to view, modify, or build the project.
    *   See the `README.md` inside this folder for technical details and setup instructions.

## How to Install the App

If you just want to try out the LibRaX application on your Android device, follow these steps:

1.  **Locate the APK**: Open the `Application` folder in this directory.
2.  **Transfer to Device**: Copy the `LibRaX (Version 1.0).apk` file to your Android device (via USB, Google Drive, email, etc.).
3.  **Install**:
    *   On your Android device, locate the file using a File Manager.
    *   Tap on the APK file to begin installation.
    *   *Note*: You may need to enable "Install from Unknown Sources" in your device settings if prompted.
4.  **Run**: Once installed, you can open LibRaX from your app drawer.
